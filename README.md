# 🔑 Secure Password Generator

A Python script that generates highly secure passwords based on user-defined length.

## 🚀 Features
* Custom Length: Generates passwords of any size based on user input.
* Input Validation: Uses ".isdigit()" to prevent system crashes from accidental text entry.
* Randomized Generation: Implements official `random` and `string` libraries for character pooling.

## 🛠️ Built With
* Python 3 (Randomization, Continuous Loops, String Libraries)
* Git & GitHub
